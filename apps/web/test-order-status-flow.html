Store Owner Actions

Available actions for each status:

  • PENDING: Confirm or Reject order
  • CONFIRMED: Add/Edit discount, Mark as Shipped
  • SHIPPED: Mark as Delivered
  • DELIVERED: No further actions (order complete)
